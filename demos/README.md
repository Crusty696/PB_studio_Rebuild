# PB Studio Demo Materials

This directory contains sales and demo materials for PB Studio.

## Directory Structure

```
demos/
├── sales_deck/          # Sales presentation materials
├── sample_videos/       # Demo videos showcasing features
├── screenshots/         # Key feature screenshots
└── README.md           # This file
```

## Sales Deck

The sales deck (`sales_deck/PB_Studio_Sales_Deck.md`) is a markdown presentation that highlights:
- Core value proposition
- Key features with visual examples
- Technical differentiators
- Use cases and target audience
- Live demo flow

To convert to PowerPoint/PDF, use tools like:
- [Marp](https://marp.app/) - Markdown to slides
- [Slidev](https://sli.dev/) - Vue-powered presentation
- [reveal.js](https://revealjs.com/) - HTML presentations

## Sample Videos

The `sample_videos/` directory contains scripts and instructions for generating:
1. **Before/After Comparison** - Manual editing vs. PB Studio auto-edit
2. **Feature Showcase** - Beat-sync capabilities in action
3. **Workflow Demo** - End-to-end editing process

## Screenshots

Key screenshots demonstrating:

## Usage

### For Sales Presentations
1. Review `sales_deck/PB_Studio_Sales_Deck.md`
2. Customize with your specific use case
3. Add screenshots from `screenshots/` directory
4. Convert to your preferred presentation format

### For Demo Videos
1. Follow instructions in `sample_videos/demo_video_guide.md`
2. Use sample footage or customer-provided content
3. Record screen capture during editing process
4. Edit final comparison video

### For Marketing Materials
- All screenshots are high-resolution and ready for web/print
- Videos are exported at 1080p with proper encoding
- Deck can be customized with company branding
